# cilicili

 >基于Flutter开发的仿哔哩哔哩安卓客户端应用，仅供学习使用

## 快速开始

- [官方插件中心](https://pub.dev/)
- [组件库](http://laomengit.com/flutter/widgets/widgets_structure.html)
- [入门](https://book.flutterchina.club/)

**新建Flutter项目**

- 1.查看——命令面板，或者Ctrl + Shift + P
- 2.输入 Flutter: New Application Project
- 3.输入项目名称，选择项目路径即可。
- 4.VS Code会自动打开一个新窗口示例。

**或者命令行运行**

```bash
flutter create my_app // 创建新的flutter项目
flutter run  // 运行程序
flutter doctor // 检查运行环境
```

**升级flutter报错**
 解决方法：使用命令：
```
flutter pub cache repair
flutter clean
```
